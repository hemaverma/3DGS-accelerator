01 Why Rust for 3DGS

The Case for Rust

Python is the right language for research iteration. But once the algorithm is understood, Rust offers a compelling production path: memory safety without GC pauses, predictable latency, native WASM compilation for browser-based viewers, and GPU access through wgpu — a cross-platform API that dispatches to Vulkan, Metal, DX12, or WebGPU from a single codebase.

Rust 3DGS — Where It Fits in the Ecosystem

Python for research & training · Rust for production viewers, real-time apps, and WASM deployment

What Rust brings

✓
Zero-cost abstractions — iterators, generics, traits compile to the same machine code as hand-written C
✓
No GC pauses — critical for real-time rendering where frame budget is fixed at 16ms
✓
wgpu — write shaders once in WGSL, run on Metal (M3), CUDA via Vulkan, WebGPU in browser
✓
burn crate provides PyTorch-like tensor ops + autograd with CUDA/Metal/CPU backends

Trade-offs vs Python

~
Longer compile times — wgpu + burn dependency tree is large. Use sccache and workspace caching
~
Smaller ML ecosystem — burn is maturing fast but PyTorch has more ops. Hybrid approach (train Python, render Rust) is common
~
Borrow checker learning curve — GPU buffer lifetimes require careful ownership design, but the compiler guides you

03 Core Data Structures

The Gaussian Scene in Rust

Rust's ownership system maps naturally to GPU buffer management. We use Structure of Arrays just like the CUDA version, but Rust's type system enforces the invariants at compile time.

gaussian_model.rs — the scene representation
Rust · src/gaussian_model.rs
use glam::{Vec3, Vec4, Quat};
use bytemuck::{Pod, Zeroable};

// ── GPU-side layout: must be #[repr(C)] for bytemuck ──────────────
// This is what actually lives in the wgpu buffer — tightly packed.
// One entry per Gaussian, but fields are in SEPARATE buffers (SoA).

#[repr(C)]
#[derive(Clone, Copy, Pod, Zeroable, Debug)]
pub struct GaussianPosition {
    pub xyz: [f32; 3],
    _pad: f32,  // 16-byte align for wgpu uniform requirements
}

#[repr(C)]
#[derive(Clone, Copy, Pod, Zeroable, Debug)]
pub struct GaussianShape {
    pub scale: [f32; 3],  // log-scale (exp to get sx,sy,sz)
    pub quat:  [f32; 4],  // quaternion (w,x,y,z)
    pub opacity: f32,      // logit-opacity
}

#[repr(C)]
#[derive(Clone, Copy, Pod, Zeroable)]
pub struct GaussianColor {
    pub sh_dc:   [f32; 3],         // degree-0 SH (RGB)
    pub sh_rest: [f32; 45],        // degrees 1-3 (15 coeffs × 3 channels)
}

// ── CPU-side GaussianScene: owns the data + GPU buffers ──────────
pub struct GaussianScene {
    // CPU storage (for densification / pruning on CPU via rayon)
    pub positions: Vec<GaussianPosition>,
    pub shapes:    Vec<GaussianShape>,
    pub colors:    Vec<GaussianColor>,

    // GPU buffers (owned, dropped when scene is dropped)
    pub buf_positions: wgpu::Buffer,
    pub buf_shapes:    wgpu::Buffer,
    pub buf_colors:    wgpu::Buffer,
    // Output buffers (written by preprocess kernel)
    pub buf_xy_screen: wgpu::Buffer,
    pub buf_cov2d:     wgpu::Buffer,
    pub buf_depths:    wgpu::Buffer,
    pub buf_rgb_eval:  wgpu::Buffer,
}

impl GaussianScene {
    pub fn new(device: &wgpu::Device, capacity: usize) -> Self {
        use wgpu::BufferUsages as U;

        // Helper: allocate a typed GPU buffer
        let make_buf = |n: usize, elem_bytes: usize, usage: U| {
            device.create_buffer(&wgpu::BufferDescriptor {
                label: None,
                size: (n * elem_bytes) as u64,
                usage: usage | U::COPY_DST | U::COPY_SRC,
                mapped_at_creation: false,
            })
        };

        let s = U::STORAGE;  // read/write in compute shaders
        Self {
            positions:    Vec::with_capacity(capacity),
            shapes:       Vec::with_capacity(capacity),
            colors:       Vec::with_capacity(capacity),
            buf_positions: make_buf(capacity, 16, s),
            buf_shapes:    make_buf(capacity, 32, s),
            buf_colors:    make_buf(capacity, 192, s),
            buf_xy_screen: make_buf(capacity, 8,  s),
            buf_cov2d:     make_buf(capacity, 12, s),
            buf_depths:    make_buf(capacity, 4,  s),
            buf_rgb_eval:  make_buf(capacity, 12, s),
        }
    }

    /// Upload CPU vectors to GPU buffers — called after densification
    pub fn upload(&self, queue: &wgpu::Queue) {
        queue.write_buffer(&self.buf_positions, 0,
            bytemuck::cast_slice(&self.positions));
        queue.write_buffer(&self.buf_shapes, 0,
            bytemuck::cast_slice(&self.shapes));
        queue.write_buffer(&self.buf_colors, 0,
            bytemuck::cast_slice(&self.colors));
    }

    pub fn len(&self) -> usize { self.positions.len() }

    /// Sigmoid of logit-opacity — used during evaluation
    pub fn opacity(logit: f32) -> f32 { 1.0 / (1.0 + (-logit).exp()) }
    /// Scale from log-space (stored) to actual metres
    pub fn scale(log_s: f32) -> f32 { log_s.exp() }
}
04 WGSL Compute Shaders

Cross-Platform Shaders via WGSL

WGSL (WebGPU Shading Language) is Rust's native shader language. wgpu compiles it to MSL on M3 Apple Silicon, SPIR-V→GLSL for Vulkan/CUDA, and WebGPU for browsers — one shader source, every platform.

Shader 1 / 2 — Preprocess: Project Gaussians to Screen
WGSL · shaders/preprocess.wgsl
// Compiled to MSL (Metal), SPIR-V (Vulkan/CUDA), or WASM (browser)
// by wgpu at runtime — zero code changes needed per platform

// ── Binding layout (matches Rust bind_group) ─────────────────────
@group(0) @binding(0) var<storage, read>       positions:  array<vec4<f32>>;
@group(0) @binding(1) var<storage, read>       shapes:     array<GaussianShape>;
@group(0) @binding(2) var<storage, read>       colors:     array<GaussianColor>;
@group(0) @binding(3) var<uniform>              camera:     CameraUniforms;
@group(1) @binding(0) var<storage, read_write> xy_out:     array<vec2<f32>>;
@group(1) @binding(1) var<storage, read_write> cov2d_out:  array<vec3<f32>>;
@group(1) @binding(2) var<storage, read_write> depth_out:  array<f32>;
@group(1) @binding(3) var<storage, read_write> rgb_out:    array<vec3<f32>>;

struct GaussianShape {
    scale:   vec3<f32>,
    opacity: f32,
    quat:    vec4<f32>,  // w,x,y,z
}

struct CameraUniforms {
    view:     mat4x4<f32>,  // world → camera
    proj:     mat4x4<f32>,  // camera → clip
    cam_pos:  vec3<f32>,
    width:    f32,
    height:   f32,
}

// Quaternion → 3×3 rotation matrix
fn quat_to_mat(q: vec4<f32>) -> mat3x3<f32> {
    let w = q.x; let x = q.y; let y = q.z; let z = q.w;
    return mat3x3<f32>(
        1.0 - 2.0*(y*y+z*z),  2.0*(x*y-w*z),      2.0*(x*z+w*y),
        2.0*(x*y+w*z),      1.0 - 2.0*(x*x+z*z),  2.0*(y*z-w*x),
        2.0*(x*z-w*y),      2.0*(y*z+w*x),      1.0 - 2.0*(x*x+y*y)
    );
}

// EWA projection: 3D covariance → 2D screen covariance
// Σ' = J · W · Σ · Wᵀ · Jᵀ  (upper triangle: a, b, c)
fn project_cov3d(Sigma: mat3x3<f32>, p_cam: vec3<f32>,
                  W: mat3x3<f32>) -> vec3<f32> {
    let tx = p_cam.x; let ty = p_cam.y; let tz = p_cam.z;
    // Jacobian of perspective divide ∂(x/z)/∂(x,y,z)
    let J = mat3x3<f32>(
        1.0/tz,  0.0,   -tx/(tz*tz),
        0.0,     1.0/tz, -ty/(tz*tz),
        0.0,     0.0,    0.0
    );
    let T      = J * W;
    let Sigma2 = T * Sigma * transpose(T);
    // Return upper triangle + low-pass filter (+0.3 diagonal)
    return vec3<f32>(Sigma2[0][0] + 0.3, Sigma2[1][0], Sigma2[1][1] + 0.3);
}

// ── Main preprocess kernel — one invocation per Gaussian ─────────
@compute @workgroup_size(256, 1, 1)
fn preprocess(@builtin(global_invocation_id) gid: vec3<u32>) {
    let idx = gid.x;
    if (idx >= arrayLength(&positions)) { return; }

    // 1. World → camera space
    let pos_w  = positions[idx].xyz;
    let pos_c  = (camera.view * vec4<f32>(pos_w, 1.0)).xyz;
    depth_out[idx] = pos_c.z;

    // 2. Build Σ = R · diag(s²) · Rᵀ
    let sh   = shapes[idx];
    let s    = exp(sh.scale);              // log → actual scale
    let R    = quat_to_mat(normalize(sh.quat));
    let S    = mat3x3<f32>(s.x*s.x,0.0,0.0,  0.0,s.y*s.y,0.0,  0.0,0.0,s.z*s.z);
    let Sigma= R * S * transpose(R);

    // 3. Project Σ → 2D screen covariance
    let W3   = mat3x3<f32>(camera.view[0].xyz, camera.view[1].xyz, camera.view[2].xyz);
    cov2d_out[idx] = project_cov3d(Sigma, pos_c, W3);

    // 4. Clip → screen pixels
    let p_clip = camera.proj * vec4<f32>(pos_c, 1.0);
    xy_out[idx] = vec2<f32>(
        (p_clip.x / p_clip.w + 1.0) * 0.5 * camera.width,
        (1.0 - p_clip.y / p_clip.w) * 0.5 * camera.height
    );

    // 5. Evaluate SH degree-0 colour (view-independent base)
    // Full SH eval omitted for brevity — same formula as CUDA/Metal
    let col = colors[idx];
    let SH_C0 = 0.28209479177;  // 1/(2√π)
    rgb_out[idx] = vec3<f32>(0.5) + SH_C0 * col.sh_dc;
}
Shader 2 / 2 — Tile Rasterizer: Alpha Composite
WGSL · shaders/rasterize.wgsl — the inner compositing loop
// Tile-based rasterizer — 16×16 workgroup = one screen tile
// WGSL workgroup_size(16, 16, 1) → 256 threads per tile

@group(0) @binding(0) var<storage, read> tile_ranges: array<vec2<u32>>;
@group(0) @binding(1) var<storage, read> sorted_ids:  array<u32>;
@group(0) @binding(2) var<storage, read> xy:          array<vec2<f32>>;
@group(0) @binding(3) var<storage, read> cov2d:       array<vec3<f32>>;
@group(0) @binding(4) var<storage, read> opacities:   array<f32>;
@group(0) @binding(5) var<storage, read> rgb:         array<vec3<f32>>;
@group(1) @binding(0) var<storage, read_write> out_image: array<vec4<f32>>;

// Workgroup-local cache (= CUDA shared memory / Metal threadgroup memory)
var<workgroup> wg_xy:    array<vec2<f32>, 256>;
var<workgroup> wg_cov:   array<vec3<f32>, 256>;
var<workgroup> wg_alpha: array<f32, 256>;
var<workgroup> wg_rgb:   array<vec3<f32>, 256>;

@compute @workgroup_size(16, 16, 1)
fn rasterize(
    @builtin(global_invocation_id)   gid:  vec3<u32>,  // pixel (x,y)
    @builtin(local_invocation_index)  lid:  u32,        // 0..255 within tile
    @builtin(workgroup_id)            wid:  vec3<u32>,  // tile (tx,ty)
) {
    let tile_idx = wid.y * grid_w + wid.x;
    let range    = tile_ranges[tile_idx];  // [start, end) of sorted_ids
    let pixel    = gid.xy;

    var C: vec3<f32> = vec3(0.0);
    var T: f32       = 1.0;

    let n_gauss = range.y - range.x;
    let rounds  = (n_gauss + 255u) / 256u;

    for (var r = 0u; r < rounds; r++) {

        // ── Cooperative load: each thread fetches one Gaussian ──────
        let load_idx = range.x + r * 256u + lid;
        if (load_idx < range.y) {
            let g      = sorted_ids[load_idx];
            wg_xy[lid]    = xy[g];
            wg_cov[lid]   = cov2d[g];
            wg_alpha[lid] = opacities[g];
            wg_rgb[lid]   = rgb[g];
        }
        workgroupBarrier();  // = __syncthreads() / threadgroup_barrier()

        // ── Each thread blends loaded Gaussians into its pixel ──────
        let batch = min(256u, n_gauss - r * 256u);
        for (var j = 0u; j < batch && T > 0.0001; j++) {
            let d   = vec2<f32>(pixel) - wg_xy[j];
            let cov = wg_cov[j];
            // Mahalanobis: power = -½ dᵀ Σ⁻¹ d
            let det   = cov.x*cov.z - cov.y*cov.y;
            let power = -0.5 * (cov.z*d.x*d.x
                               - 2.0*cov.y*d.x*d.y
                               + cov.x*d.y*d.y) / det;
            if (power > 0.0) { continue; }  // outside Gaussian

            let alpha = min(0.99, wg_alpha[j] * exp(power));
            if (alpha < 1.0/255.0) { continue; }

            C  += wg_rgb[j] * alpha * T;
            T  *= (1.0 - alpha);
        }
        workgroupBarrier();
    }

    if (pixel.x < u32(camera.width) && pixel.y < u32(camera.height)) {
        let pix = pixel.y * u32(camera.width) + pixel.x;
        out_image[pix] = vec4<f32>(C + T * bg_color, 1.0);
    }
}

05 Training with Burn

Differentiable Training via burn

The burn crate provides PyTorch-like tensor operations with autograd. The burn-wgpu backend dispatches to Metal on M3 or Vulkan on NVIDIA — same code for both.

Training loop — device-agnostic burn tensors
Rust · src/train.rs
use burn::{
    backend::{Autodiff, Wgpu},   // Wgpu = Metal on M3, Vulkan on NVIDIA
    optim::{Adam, AdamConfig},
    tensor::{Tensor, TensorData},
};

// Backend type alias — swap Wgpu for burn_cuda::Cuda for CUDA path
type Backend  = Autodiff<Wgpu>;
type BkInner = Wgpu;

pub struct Trainer {
    // Gaussian parameters as burn Tensors with gradients
    xyz:         Tensor<Backend, 2>,  // [N, 3]
    log_scale:   Tensor<Backend, 2>,  // [N, 3]
    quat:        Tensor<Backend, 2>,  // [N, 4]
    logit_alpha: Tensor<Backend, 2>,  // [N, 1]
    sh_dc:       Tensor<Backend, 3>,  // [N, 1, 3]
    sh_rest:     Tensor<Backend, 3>,  // [N, 15, 3]
    // Separate Adam optimizer per param group (different LR per param)
    opt_xyz:     Adam<Backend>,
    opt_shape:   Adam<Backend>,
    opt_color:   Adam<Backend>,
}

impl Trainer {
    pub fn new(pcd: &PointCloud) -> Self {
        let n = pcd.len();
        // Tensor::from_data initialises on the wgpu device (Metal / Vulkan)
        Self {
            xyz: Tensor::from_data(TensorData::from(pcd.points.clone()),
                                    &Default::default()),
            log_scale: Tensor::full([n, 3], -3.0, &Default::default()),
            quat: {
                // Identity quaternion w=1, x=y=z=0
                let mut q = Tensor::zeros([n, 4], &Default::default());
                q = q.slice_assign([0..n, 0..1], Tensor::ones([n,1], &Default::default()));
                q
            },
            logit_alpha: Tensor::full([n, 1], -2.2, &Default::default()),
            sh_dc:   Tensor::zeros([n,  1, 3], &Default::default()),
            sh_rest: Tensor::zeros([n, 15, 3], &Default::default()),
            // Different learning rates per parameter group
            opt_xyz:   AdamConfig::new().with_lr(1.6e-4).init(),
            opt_shape: AdamConfig::new().with_lr(5e-3).init(),
            opt_color: AdamConfig::new().with_lr(2.5e-3).init(),
        }
    }

    pub fn step(&mut self, rendered: Tensor<Backend, 3>,
                          gt:       Tensor<Backend, 3>) -> f32 {
        // ── Loss: L1 + SSIM ──────────────────────────────────────────
        let l1   = (rendered.clone() - gt.clone()).abs().mean();
        let ssim = ssim_loss(rendered.clone(), gt);
        let loss = l1 * 0.8 + ssim * 0.2;

        let loss_val = loss.clone().into_scalar();

        // ── Backprop: burn's autodiff computes all gradients ──────────
        let grads = loss.backward();

        // ── Update each parameter group with its optimizer ────────────
        let xyz_grad  = Tensor::from_inner(grads.get(&self.xyz).unwrap());
        self.xyz = self.opt_xyz.step(self.xyz.clone(), xyz_grad);

        // (similar for log_scale, quat, logit_alpha, sh_dc, sh_rest)

        loss_val
    }
}
06 Densification

Clone · Split · Prune in Rust

Densification runs on the CPU using rayon for parallel iteration, then uploads the new Gaussian list back to GPU buffers. Rust's ownership model makes it impossible to accidentally use a stale GPU buffer — the compiler enforces it.

densify.rs — adaptive density control
Rust · src/densify.rs
use rayon::prelude::*;

pub struct DensifyConfig {
    pub grad_threshold:  f32,   // 0.0002  — trigger if grad > this
    pub size_threshold:  f32,   // 0.01   — large if max_scale > this
    pub prune_opacity:   f32,   // 0.005  — remove if opacity < this
    pub split_scale_factor: f32, // 1.6    — split children scale
}

impl GaussianScene {
    /// Called every 100 training steps. Returns number of changes made.
    pub fn densify_and_prune(
        &mut self,
        grad_norms: &[f32],   // [N] gradient magnitude per Gaussian
        cfg: &DensifyConfig,
    ) -> usize {
        let n = self.len();

        // ── Phase 1: Identify candidates in parallel ─────────────────
        let actions: Vec<Action> = (0..n)
            .into_par_iter()          // rayon — runs on all CPU cores
            .map(|i| {
                let grad = grad_norms[i];
                let sh   = &self.shapes[i];
                let max_s = sh.scale.iter()
                              .map(|&s| s.exp())
                              .fold(0.0f32, f32::max);
                let opacity = GaussianScene::opacity(sh.opacity);

                if opacity < cfg.prune_opacity {
                    Action::Prune
                } else if grad > cfg.grad_threshold {
                    if max_s > cfg.size_threshold {
                        Action::Split   // large + high grad → split
                    } else {
                        Action::Clone   // small + high grad → clone
                    }
                } else {
                    Action::Keep
                }
            })
            .collect();

        // ── Phase 2: Apply actions (sequential to avoid index confusion)
        let mut new_pos:   Vec<GaussianPosition> = Vec::new();
        let mut new_shape: Vec<GaussianShape>    = Vec::new();
        let mut new_color: Vec<GaussianColor>    = Vec::new();

        let mut changed = 0usize;

        for (i, action) in actions.iter().enumerate() {
            match action {
                Action::Keep | Action::Clone => {
                    new_pos.push(self.positions[i]);
                    new_shape.push(self.shapes[i]);
                    new_color.push(self.colors[i]);
                    if matches!(action, Action::Clone) {
                        // Duplicate with small random offset in gradient direction
                        let mut clone_pos = self.positions[i];
                        clone_pos.xyz[0] += tiny_rand() * 0.01;
                        clone_pos.xyz[1] += tiny_rand() * 0.01;
                        clone_pos.xyz[2] += tiny_rand() * 0.01;
                        new_pos.push(clone_pos);
                        new_shape.push(self.shapes[i]);
                        new_color.push(self.colors[i]);
                        changed += 1;
                    }
                }
                Action::Split => {
                    let sh = self.shapes[i];
                    let pos = self.positions[i];
                    // Scale both children down by split factor
                    let mut child_shape = sh;
                    for s in &mut child_shape.scale {
                        *s -= cfg.split_scale_factor.ln();  // log-space /= factor
                    }
                    // Two children offset along principal axis
                    let axis = principal_axis_from_quat(sh.quat);
                    let r    = sh.scale[0].exp() * 0.5;
                    for sign in [1.0f32, -1.0] {
                        let mut child_pos = pos;
                        child_pos.xyz[0] += sign * axis[0] * r;
                        child_pos.xyz[1] += sign * axis[1] * r;
                        child_pos.xyz[2] += sign * axis[2] * r;
                        new_pos.push(child_pos);
                        new_shape.push(child_shape);
                        new_color.push(self.colors[i]);
                    }
                    changed += 1;
                }
                Action::Prune => { changed += 1; }  // skip = pruned
            }
        }

        self.positions = new_pos;
        self.shapes    = new_shape;
        self.colors    = new_color;
        changed
    }
}

#[derive(Clone, Copy)]
enum Action { Keep, Clone, Split, Prune }

pub struct GaussianRenderer {
    device:           Device,
    queue:            Queue,
    preprocess_pipe:  ComputePipeline,
    rasterize_pipe:   ComputePipeline,
    sort_pipe:        ComputePipeline,  // bitonic sort in WGSL

impl GaussianRenderer {
    pub async fn new() -> Self {
        // 1. Request the best available GPU adapter
        let instance = Instance::new(InstanceDescriptor {
            // Backends::all() → picks Metal on macOS, Vulkan on Linux/Win
            backends: Backends::all(),
            ..Default::default()
        });
        let adapter = instance
            .request_adapter(&RequestAdapterOptions {
                power_preference: PowerPreference::HighPerformance,
                ..Default::default()
            })
            .await.unwrap();

        // Print: "Metal" on M3, "Vulkan" on NVIDIA
        println!("Backend: {:?}", adapter.get_info().backend);

        let (device, queue) = adapter
            .request_device(&DeviceDescriptor {
                required_limits: Limits::downlevel_defaults(),
                ..Default::default()
            }, None)
            .await.unwrap();

        // 2. Compile WGSL shaders (→ MSL on Metal, SPIR-V on Vulkan)
        let load_shader = |src: &str| {
            device.create_shader_module(ShaderModuleDescriptor {
                label: None,
                source: ShaderSource::Wgsl(src.into()),
            })
        };
        let pre_shader  = load_shader(include_str!("../shaders/preprocess.wgsl"));
        let ras_shader  = load_shader(include_str!("../shaders/rasterize.wgsl"));
        let sort_shader = load_shader(include_str!("../shaders/bitonic_sort.wgsl"));

        let make_pipe = |shader: &ShaderModule, entry: &str| {
            device.create_compute_pipeline(&ComputePipelineDescriptor {
                layout: None,
                module: shader,
                entry_point: entry,
                ..Default::default()
            })
        };

        Self {
            preprocess_pipe: make_pipe(&pre_shader,  "preprocess"),
            rasterize_pipe:  make_pipe(&ras_shader,  "rasterize"),
            sort_pipe:       make_pipe(&sort_shader, "bitonic_sort"),
            device, queue,
        }
    }

    pub fn render(&self, scene: &GaussianScene, cam: &Camera,
                   out: &wgpu::Buffer) {
        let n   = scene.len() as u32;
        let mut encoder = self.device.create_command_encoder(&Default::default());

        // ── Pass 1: Preprocess (project + SH eval) ────────────────────
        {
            let mut pass = encoder.begin_compute_pass(&Default::default());
            pass.set_pipeline(&self.preprocess_pipe);
            pass.set_bind_group(0, &scene.input_bind_group, &[]);
            pass.set_bind_group(1, &scene.output_bind_group, &[]);
            pass.dispatch_workgroups((n + 255) / 256, 1, 1);
        }  // pass dropped → commands recorded, barrier inserted

        // ── Pass 2: Sort by depth (GPU bitonic sort in WGSL) ─────────
        {
            let mut pass = encoder.begin_compute_pass(&Default::default());
            pass.set_pipeline(&self.sort_pipe);
            pass.set_bind_group(0, &scene.sort_bind_group, &[]);
            // Bitonic sort: log²(N) passes
            let log2n = (n as f32).log2().ceil() as u32;
            for _ in 0..log2n * (log2n + 1) / 2 {
                pass.dispatch_workgroups(n / 512, 1, 1);
            }
        }

        // ── Pass 3: Tile rasterize (alpha composite) ──────────────────
        {
            let mut pass = encoder.begin_compute_pass(&Default::default());
            pass.set_pipeline(&self.rasterize_pipe);
            pass.set_bind_group(0, &scene.render_input_bg, &[]);
            pass.set_bind_group(1, &scene.render_output_bg, &[]);
            // One workgroup per 16×16 tile
            pass.dispatch_workgroups(
                (cam.width  + 15) / 16,
                (cam.height + 15) / 16,
                1,
            );
        }

        self.queue.submit([encoder.finish()]);
        // queue.submit returns immediately — GPU runs async
        // Use device.poll(Maintain::Wait) to synchronise if needed
    }
}
